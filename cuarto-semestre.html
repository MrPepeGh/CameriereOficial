Avances Técnicos - Cuarto Semestre

En esta etapa del proyecto, el enfoque se ha centrado en la escalabilidad, la eficiencia de los datos y la experiencia de usuario en tiempo real.

Gestión en Vivo (WebSockets)

Se integró el protocolo WebSocket para transformar la comunicación unidireccional en una experiencia bidireccional constante. Esto permite que el flujo de órdenes entre la mesa, la cocina y la administración ocurra de forma instantánea sin necesidad de recargar la página, garantizando una actualización de pedidos en tiempo real indispensable para entornos de alta demanda.

Lógica de Estructuras

Aplicamos fundamentos avanzados de Estructuras de Datos mediante la implementación de listas dinámicas para la organización de la información. Esta lógica permite un manejo de memoria más eficiente y una manipulación de colecciones de objetos (como productos y categorías) mucho más robusta, optimizando los algoritmos de búsqueda y ordenamiento dentro del sistema.

Migración a PostgreSQL

Con el objetivo de preparar el proyecto para un entorno de producción, se realizó la migración de la base de datos a PostgreSQL. Este cambio asegura una mayor integridad referencial y soporte para consultas complejas, facilitando un despliegue online estable y aprovechando las capacidades de concurrencia que este motor de base de datos relacional ofrece.
